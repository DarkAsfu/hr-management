
## Installation

To run this project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/DarkAsfu/Shreyu.git

2. Navigate to the project directory:
   
   ```bash
   cd Shreyu

3. Install dependencies:
   
   ```bash
   npm install

4. Run project
    
    ```bash 
    npm run dev
